# 재개발 투자 판단 시스템 (redev)

임의의 주택 주소를 입력하면 그 일대가 **(1) 재개발될 환경인가 (2) 되면 얼마나 오를
입지인가 (3) 언제쯤 (4) 무산·정체 리스크는 (5) 진입 가능한가**를 후보 경계·근거와 함께
지도로 제시하는 **투자 판단 도구**. 상세 설계 원칙은 [CLAUDE.md](CLAUDE.md).

> ⚠️ 투자 권유가 아니라 데이터 기반 참고치 + 근거 제공. 가치·수익 출력은 항상 추정·범위·
> 불확실성을 명시한다([CLAUDE.md §9](CLAUDE.md)).

## 두 개의 AI 심장

- **심장 1 — 구역 분할(GNN)**: 필지 인접 그래프 위 노드 분류로 "재개발 환경" 후보 경계 학습.
- **심장 2 — AVM(Stage 3)**: 실거래가로 입지가치·상승여력 학습. *(Phase 5)*

## 현황 (v1, 서울 4구: 성북·동작·은평·구로)

**Phase 0~3 완료** — 데이터 파이프라인 + 심장1 노드 분류 + 정직한 평가.

### ★Phase 3 핵심 결과 — R9 통제 비교 (심장1)

`(필지, 시점)` 시점 라벨 + per-t 이웃집계 + **leave-one-district-out 공간 CV**(R3)로
대조군 사다리를 채점:

| 모델 | PR-AUC | 격전지 recall (aging=0 양성) |
|---|---|---|
| B-2 (aging 임계, 동어반복 바닥선) | 0.640 | 0.000 |
| B1 (XGBoost + 1홉 이웃집계) | 0.932 | 0.631 |
| B1+ (XGBoost + 2홉 이웃집계) | 0.934 | 0.622 |
| GNN (GraphSAGE 2층) | 0.929 | 0.606 |
| GNN + 자기지도 사전학습 | 0.928 | 0.616 |

**판정**: GNN은 베이스라인 천장(B1+ 0.934)을 **넘지 못하고 미세하게 뒤진다.** zone-block
bootstrap(양성 구역 + 음성 공간블록 모두 resample, n=1000): Δ(B1+ − GNN+pretrain) PR-AUC
중앙 **+0.006**, 95% CI **[+0.002, +0.012]** — 0 불포함, **통계적으로 유의하나 실질 효과크기는
무시 가능**. "능가" 증거는 전무하고, 구로·은평 fold 열세·성북·동작 동률. 자기지도 사전학습
리프트도 ≈0(PR-AUC **−0.001**) — **v1 피처 5차원에선 사전학습 무용**을 측정으로 확정.

이는 *모델 실패가 아니라 **데이터 병목***(R9): 유효표본 42구역 + 피처 5차원으로는 학습된
메시지 패싱이 1홉 손집계 위에 값을 더하지 못한다. **구조보다 도달범위, 도달범위보다 데이터.**

> **한 줄**: 구조보다 도달범위, 도달범위보다 데이터.

### v1 추론 모델 = **B1 (XGBoost + 1홉 이웃집계)**

가장 단순하면서 격전지 recall 최고(0.631). 추론·경계 산출은 B1로 간다.

### GNN은 보존 — v2 재도전 조건

GNN 코드(`redev/models/gnn/`)는 **삭제하지 않고 보존**한다. 아래 조건이 충족되면 재평가:

1. **전역 라벨 확장 (R7-①)**: 서울 전역으로 positive 구역 확대 → 유효표본 42 → 수백.
2. **피처 보강**: 용도지역·공시지가·역세권 거리 등 데이터대기 피처 추가(5차원 → 확장).
3. 위 둘 후 같은 LODO/지표로 B1 대비 재평가 — 그때 구조의 값이 드러나는지 측정.

(진짜 사전학습·GNN의 무대는 v1이 아니라 v2다. v1은 그 사실을 *측정으로* 확정했다.)

```
redev/
├── config/        # 자치구·임계값(법 출처)·CV 파라미터 (매직넘버 금지)
├── data/          # ingest(소스별) · pnu(단일키) · labels((필지,시점) 키스톤) · aging(시점 노후도)
├── graph/         # build(인접 그래프) · features(시점 노드 피처)
├── models/        # baseline(B-2/B1/B1+) · gnn(model/pretrain/train) · avm·feasibility(Phase5)
├── eval/          # spatial_cv(LODO+버퍼) · metrics(PR-AUC·격전지·hard/easy)
└── ...            # docs/design(설계노트) · LEARNING_LOG.md
```
