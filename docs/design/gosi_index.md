# 설계 노트 — 고시목록 인덱스(공공데이터, 합법 A축) (v1_3-gosi)

> 규칙1: 코드 전에 이 노트. 규칙9: ToS 검토 완료(본문 자동스크래핑 보류, 합법 API/CSV로 전환).

## 0. 한 줄
국토교통부 **토지이음 고시목록**(data.go.kr 15083101, 이용허락 제한 없음·상업가능·무료)에서
`고시번호 → 공식 접속 URL·고시일·고시명`을 인덱싱한다. 본문은 이 URL로 *사람이* 수동 다운로드
(C, ToS 안전). 자동화의 **합법 1축** + 인덱스 토대.

## 1. ★측정·ToS 판정 (2026-06-14)
- 토지이음 robots = Yeti(네이버)만 명시 / 정보몽땅 robots 없음 / 약관 본문 미검증 → **본문 자동
  스크래핑은 '허용' 확인 불가 → 보류**(상업 전제, 합법 우선).
- 합법 대안: 15083101 = CSV+OpenAPI, 컬럼 [고시번호·고시일자·접속링크주소(URL)·고시명], 전국 448,101행,
  **갱신주기 연간**(다음 2026-06-30). → ★인덱스·URL은 합법 확보, 단 신규 감지엔 '연간'이 약점(§6 백로그).

## 2. 입력 → 출력
- 입력: (a) 로컬 CSV(사용자가 data.go.kr에서 다운, 무로그인) 또는 (b) Open API(serviceKey, 로컬 실행).
- 출력: 정규화 인덱스 `[고시번호, 고시일자, url, 고시명]` + `urls_for(고시번호 리스트) → 다운로드 매니페스트`.
- ★우리 zone의 고시번호(기존 결정고시 CSV)와 조인 → 우선순위 13개의 **URL 목록**(클릭 다운로드용).

## 3. 함수/파일 분해
```text
data/ingest/gosi_index.py
├── load_index(csv_path)                 # 로컬 CSV → 정규화 df (무키, 즉시 가능)
├── fetch_index_api(*, service_key, endpoint, rows, max_pages)   # data.go.kr 표준 패턴(로컬 실행)
└── urls_for(고시번호_list, index)        # 고시번호 → (고시번호, 고시명, url) 매니페스트
scripts/_gosi_urls.py                     # 우선순위 13개 고시번호 → URL 출력(사용자가 CSV/API 확보 후)
```

## 4. data.go.kr API — 표준 패턴 (엔드포인트는 사용자 발급 후 확정)
- data.go.kr OpenAPI 공통: `GET <요청주소>?serviceKey=&pageNo=&numOfRows=&type=json` →
  `{response:{header:{resultCode,resultMsg}, body:{items:[...], totalCount, pageNo, numOfRows}}}`.
- ★정확한 요청주소(operation)는 **활용신청 후 '활용가이드/엔드포인트' 페이지**에 표기 → config/인자로 받는다
  (코드에 하드코딩 금지, 규칙5). serviceKey는 .env(`DATA_GO_KR_KEY`), 절대 커밋 금지.
- 페이지네이션: numOfRows(예 1000) × pageNo 루프, totalCount까지. rate limit 절제(간격·재시도).

## 5. ★실행 경계
- ★Claude Code 환경은 외부접속 차단 → API 호출·CSV 다운로드는 **사용자 로컬 실행**. 코드만 작성.
- CSV 경로는 키 없이 즉시(사용자가 파일만 받아 두면 파싱·조인은 로컬/CI 어디서나).

## 6. ★갱신 백로그 (근거 남김)
- 15083101 갱신 = **연간** → 신규 고시 실시간 감지엔 부적합. '데이터 썩음' 핵심 해결엔 더 빠른 합법
  소스 필요: 서울시보(주간)·서울 열린데이터 결정고시 갱신주기 확인·정보몽땅 RSS(있으면).
- **B(제공신청) 조사**: 서울시/국토부에 정비계획 **결정조서**(용적률·세대수 구조화) 제공신청 가능 여부 —
  부산(정비사업 데이터: 용적률·건폐율·세대수 공개) 선례 있음. 되면 PDF 파싱 불필요(가장 깨끗). 상업 출시
  준비 시 정식 신청, 지금은 근거만 기록(§DATA_SOURCES).
