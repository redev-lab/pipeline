# 설계 노트 — v1.1 피처 사이클 (직교 축 + 재경기)

> 규칙1: 코드 전에 이 노트. 승인 후 구현(한 모듈씩). CLAUDE.md §5(R1·R9)·labels §9·§13 우선.

## 0. 한 문단 요약

입고·검수완료 데이터 3종(공시지가 연도별·용도지역·역사마스터)으로 **노후도 외 직교 축**을
피처에 넣고(5→10+차원), 두 심장+infer를 **같은 평가 장비(LODO·같은 지표·zone-block bootstrap)**
로 재측정한다. 목적은 "데이터병목"(심장1·2·infer 공통 판정)이 피처 보강으로 풀리는지 *측정*.
★시점정합(R1)이 사활: 학습 피처는 라벨 t에 맞춰, 추론은 2026.

## 1. ingest 3종 (시점정합 규율)

### 1-1. 개별공시지가 (연도별 2001~2026) — 가치 축
- **조인**: ★토지코드 = PNU 19자리(직접). 19자리 아닌 행 drop(검수: 2010년 ~575행).
- **시점정합**: 학습 피처 = ★라벨 행의 t에 맞는 **연도 값(as-of-t)**, 추론 = 2026. (연도별 파일이라
  R1 누수 없음 — positive 2008은 2008 공시지가.)
- **상대화**: 절대 원/㎡ 대신 ★**(연도, 자치구) 백분위**(labels §9) — 시대·지역 시세차 오염 완화.
- 결측 필지: 공시지가 없는 PNU는 백분위 결측 → 중앙(0.5) 대치 + 결측 플래그 피처.

### 1-2. 용도지역 (AL_D124, 2026 스냅샷) — 입지/규제 축
- **조인**: 필지 대표점 ⨯ 용도지역 폴리곤 **공간조인**(EPSG:5186 일치, 변환 불요). A6=세분류명.
- **피처**: 세분류 카테고리(제1/2/3종일반주거·준주거·상업·녹지…) → ordinal(주거밀도 순) 또는
  one-hot 소수. 정석: stage1 저밀이용을 **법정용적률 대비**로(절대 FAR 근사 교체).
- ★**2026 스냅샷 한계(§13 등록)**: 과거 라벨에 붙이면 종상향 역류 가능(2026 3종인데 지정 당시
  2종). **근사 수용 + 한계 명시 + ★leakage_ablation 필수**(이 피처 빼고/넣고 점수 비교 = 수검).

### 1-3. 역세권 거리 (역사마스터) — 입지 축
- **변환**: WGS84(위경도) → EPSG:5186. 필지 centroid → **최근접 역 거리**(cKDTree). 전국 역이라
  4구 인근만 자동 선택.
- ★**개통연도 소스 부재(조사 확정)**: 서울교통공사 API·자치구별·역사마스터 모두 개통일 없음.
  → v1.1 = **현재(2026) 역 기준 근사 + 한계 명시 + ablation**. 영향 크면 개통연도 확보 v2 승격.
  (line-year 추정 매핑은 '사실'인 척 금지 — 정직성.)

## 2. 피처 행렬 재조립 (features.py 확장)

- 5차원(aging·area·compactness·bldg_density·road_abut) → **+공시지가백분위·용도지역(ord)·역거리
  ·(공시지가결측 플래그) = 9~10차원**. FEATURE_COLUMNS 확장.
- ★이웃집계 재계산: 신규 피처도 per-t neighbor_aggregate(B1/B1+ 동일 규약). nb1·nb2 차원 증가.
- 전 노드 캐시(infer_features) 갱신. 학습행렬·캐시 재빌드.

## 3. 재경기 — 평가 장비 재사용 (신구 나란히)

- **노드 사다리**: B-2/B1/B1+/GNN(+pretrain) 같은 LODO·같은 지표·zone-block bootstrap 재실행.
  관전: GNN vs B1+ 격차 변화·격전지 recall·은평/구로 fold. **신구 점수표 병기**.
- **AVM**: fit_avm vs 구평균 베이스라인 재대결(가치 피처 생겼으니 — 이번엔 이기나?).
- **infer**: 81% 과대예측이 내려가나 + IoU 4종 재실행.
- **stage1**: 저밀이용 절대FAR → 용도지역 법정용적률 정석으로 교체.

## 4. ★수검 (규칙9, 구현 후)

1. **신규 피처 위생**: 결측률·범위·★조인율(공시지가 PNU 매칭·용도지역 공간조인 커버리지·역거리 분포).
2. **★leakage_ablation**: 용도지역·역거리 각각 빼고/넣고 LODO 점수 비교 — 시점정합 한계의 *영향 측정*
   (eval/leakage_ablation.py). 영향 크면 그 피처의 누수 위험을 명시·대응.
3. **신구 점수표 비교**: v1(5피처) vs v1.1(10피처) 한 표. 두 심장·infer 모두. 좋아졌으면 "병목=피처"
   확증, 안 좋아졌으면 그것도 측정(피처가 답이 아니거나 더 필요).
4. **시점정합 증명**: 공시지가가 라벨 t로 들어갔는지(positive 2008 → 2008 공시지가, 누수 0).

## 5. 검토했지만 버린 대안

- **공시지가 절대값 학습**: 시대·지역 오염 → (연도,구) 백분위.
- **용도지역 과거이력 무시하고 2026 그대로**: 종상향 역류 → 수용하되 ablation으로 영향 계량(은폐 안 함).
- **역 개통연도 추정 매핑**: 추정을 사실로 = 정직성 위반 → 현재 근사 + ablation, 소스 확보 v2.
- **새 평가 장비**: 재경기는 *같은* 장비라야 신구 비교가 공정 — 기존 LODO·bootstrap 재사용.
