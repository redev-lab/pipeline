# 공공데이터 제공신청서 — 정비구역 경계 (상업 라이선스 요청, 초안)

> 제출처: 공공데이터포털(data.go.kr) **데이터요청 > 제공신청**. 처리 10일 내 회신(+10일 연장 가능),
> 거부 시 60일 내 분쟁조정. 아래 문안을 복사해 제출(★[ ] 부분만 본인 정보로 채움).

---

**대상기관**: 서울특별시 (도시계획국 / 공간정보담당관)
　※ 전국 단위로 신청 시 → **국토교통부 (국가공간정보센터 / 공간정보정책과)** 로 기관명만 교체

**신청인**: [성명 또는 사업자명] · [연락처] · [이메일]

## 1. 신청 데이터명
서울특별시 **정비구역(주택재개발·재건축 등) 경계 공간정보**(정비구역계 폴리곤)

## 2. 데이터 상세 요청
- **범위**: 서울특별시 전역 — 현재 유효 + **과거 지정 구역 포함**(지정 이력 전체. 의제처리만이 아닌 일반 주택재개발 포함)
- **속성**: 구역명, 정비사업 유형(재개발/재건축 등), 지정 고시번호·고시일자, 자치구, (가능 시) 사업 단계
- **형식**: SHP 또는 GeoJSON (좌표계 EPSG:5186 또는 5174 명시)
- **갱신**: 최신 1회 + (가능 시) 정기 갱신본

## 3. 활용 목적 (상업)
부동산·도시계획 **데이터 분석 서비스 개발 및 상업적 제공**. 정비구역 경계를 입지·환경 분석에 활용해
일반 이용자에게 데이터 기반 참고 정보를 제공(투자 자문이 아닌 참고치 제공).

## 4. ★라이선스 요청 (핵심)
본 데이터를 **공공누리 제1유형(출처표시) 등 상업적 이용이 가능한 조건**으로 제공해 주시기를 요청합니다.
- 현재 공개된 정비구역 경계 데이터 — **서울 열린데이터광장 의제처리구역(OA-20957)은 공공누리 제4유형,
  V-World 연속주제(도시및주거환경정비/정비구역)는 비영리 조건** — 는 **상업적 이용이 제한**되어 상업 서비스에
  활용할 수 없습니다.
- 정비구역 경계는 행정기관이 고시한 공적 정보로 **제3자 저작권·개인정보를 포함하지 않으므로**, 상업 이용
  가능 조건 제공이 가능하다고 판단됩니다.
- 제1유형 일괄 전환이 어려우시면, **본 신청인에 대한 별도의 상업적 이용허락**(공공누리 안내상 가능)으로라도
  제공을 요청드립니다.

## 5. 법적 근거
- 공공데이터법 **제3조**(공공데이터 제공·이용 활성화, 영리 목적 이용 보장)
- 공공데이터법 **제27조**(공공데이터의 제공신청)
- 정비구역 경계는 정보공개법 제9조 비공개 대상·제3자 권리에 해당하지 않음

## 6. 희망 회신
제공 가능 여부, **부여 라이선스 유형**, 제공 방식(다운로드/API). 거부 시 구체적 사유.
