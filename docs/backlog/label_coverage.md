# 라벨 커버리지 한계 — 측정·백로그 (2026-06-19)

> 계기: 정릉동 170-1(정릉골/정릉3동 주택재개발, 실제 지정)을 시스템이 "지정 아님"으로 판정.
> 원인은 모델이 아니라 **pos 라벨 원천(의제처리 UQ181)이 서울 전체 지정구역의 부분집합**이라는 데이터 한계(R7).

## 1. 커버리지 실태 (학습 4구: 성북·동작·은평·구로)

| 항목 | 수치 | 비고 |

| 우리 라벨(pnu_zone) 지정 zone | **46** (성북15·동작12·은평16·구로3) | 의제처리 UQ1221/1222 → 시점·필지조인·dedup 통과분 |

### 두 층위의 누락
- **(A) 원천 내 손실 (~73→46)**: 의제처리 재개발 폴리곤조차 ~37%가 라벨에서 탈락 — 시점 미상·필지 미교차·완공/철거 오염(R2)·분합필(R11). 일부는 정당한 drop(오염 차단), 일부는 복구 여지.
- **(B) 원천 밖 누락 (근본)**: **의제처리(UQ181)에 *아예 없는* 구역** — older 일반 주택재개발이 정상 지정됐으나 '의제처리' 법적 범주가 아니라 미수록.
  - UQ181 4구 title에 **0건**으로 확인된 실제 지정구역: **정릉3동(정릉골)·장위3·장위14·돈암6·안암2·신길음1·동소문2·성북2** (jeonbisaeop 2021 교차로 식별, 더 있을 것).
  - 반면 노량진·흑석1/11 등은 UQ181에 **있어 커버됨**(초기 구역명 매칭의 false-miss였음 — 단순 이름매칭 커버리지%는 신뢰 말 것).

### A. 공공데이터 제공신청 (공공데이터법 27조)
- 창구: 공공데이터포털 **데이터요청>제공신청**(data.go.kr/tcs/dor). 대상기관=서울시/국토부.
- 처리: ★**10일 내 결정(+10일 연장)**. 거부 시 **60일 내 분쟁조정** 신청 가능. → 빠르고 저비용(신청서 1장).
- ★법리 한계: "상업금지 공공저작물은 영리 이용 불가. **다만 별도의 이용허락을 받아 상업 이용 가능**"(공공누리 안내).
  즉 제공받아도 라이선스는 기관 재량(4유형 가능) — **상업 자동 보장 X**. 단 *별도 상업 이용허락 요청* 경로는 법적으로 존재.
- 선례: 정비구역 경계가 **공공누리 1유형(상업가능)으로 풀린 명확한 선례 = 못 찾음(불명확)**. 부산 등도 미확인.
- 평가: **저비용·빠름·밑져야 본전**이나, 1유형 선례 부재 + 기관 재량 → **성공 확률 낮음~미지수, 의존 불가**.

## 6. staleness 정정 — 라벨은 안 묵었다 (2026-06-22 확인)

- ★검토자 우려("의제처리 2023-10 스냅샷, 2.5년 묵음")는 **혼동**. 우리 라벨 원천 = `_SRC["uq"]` =
  **UQ181_의제처리구역_202602(2026-02)** — 고시일 ~2026-01-08 포함, **2025년 이후 지정 348건**. 약 4개월 신선.

- 의제처리 최신본: nsdi UQ181 연속주제는 수시/월 갱신. 더 최신본 나오면 SHP 1파일 교체(스키마 동일)로 끝 — 4개월차라 비긴급.
- 고시목록(15083101): ★**2025-06-16이 최신 발행본(2026판 미발행)** → 종암9·본동 등 '얼마칸 미발견'은 갱신 전까지 그대로(수동 입수 또는 대기 백로그).
- 라이선스: 어느 버전이든 4유형(상업금지) 동일 — 갱신≠상업가능. 상업은 §5(제공신청/자체구축).
- ★결론: **라벨 최신화 불필요(이미 최신).** 정릉골 'false 지정아님'은 staleness 아닌 **커버리지**(의제처리 ⊊ 전체
  정비구역, §1-B) — 최신본으로 당겨도 정릉골(일반 주택재개발)은 안 잡힘. 근본 해결은 v2 자체구축뿐.
